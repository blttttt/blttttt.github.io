  • 操作系统：Windows 11

  • Git

安装Node

Node.js
下载后傻瓜式安装即可
安装完成后 win+R输入cmd，回车，打开命令行窗口，输入node -v，看到版本信息，则安装成功。

修改npm源。npm下载各种模块，默认从国外服务器下载，这里改配置为淘宝镜像。

安装Git

Git
下载后傻瓜式安装即可

在电脑的应用里看到有Git Bash即可

  • Git CMD 是windows 命令行的指令风格
  • Git Bash 是linux系统的指令风格（建议使用）
  • Git GUI是图形化界面（新手学习不建议使用）

打开Git Bash如图即安装完成
